# 스포츠센터 회원 관리 프로그램 - 기능 명세서

| 구분                   | 주 기능                | 상세 기능             | 설명                                                                      | 가능 여부 |
| :--------------------- | :--------------------- | :-------------------- | :------------------------------------------------------------------------ | :-------: |
| **1. 회원 관리**       | **회원 등록 (Create)** | 회원 정보 입력        | 이름(필수), 연락처(필수), 성별, 생년월일, 만료일 등 입력 후 DB 저장       | 확인 완료 |
|                        |                        | 유효성 검사           | 필수 항목 누락 시 경고창 팝업 및 날짜 입력 포맷 검증                      | 확인 완료 |
|                        | **회원 조회 (Read)**   | 전체 회원 목록 조회   | 프로그램 실행 시 전체 회원 데이터를 불러와 `JTable`에 목록 표시           | 확인 완료 |
|                        |                        | 회원 검색             | '이름' 또는 '연락처' 일부만 입력해도 일치하는 회원을 실시간 필터링        | 확인 완료 |
|                        | **회원 수정 (Update)** | 회원 정보 바인딩      | 목록에서 특정 회원 선택 시, 입력 폼에 기존 정보를 자동으로 채움           | 확인 완료 |
|                        |                        | 정보 업데이트         | 변경된 데이터를 반영하여 DB 내 해당 레코드를 갱신(`UPDATE`)               | 확인 완료 |
|                        | **회원 삭제 (Delete)** | 삭제 확인 프로세스    | 실수로 인한 삭제 방지를 위해 '정말 삭제하시겠습니까?' 확인 창 띄움        | 확인 완료 |
|                        |                        | 레코드 영구 삭제      | 사용자 최종 확인 후 DB에서 선택된 회원 데이터를 완전 삭제(`DELETE`)       | 확인 완료 |
| **2. 필터링 및 강조**  | **부가 기능**          | 만료 회원 시각적 강조 | 멤버십 종료일 7일 이내 또는 만료된 회원의 행 배경색/글자색을 다르게 표시  | 확인 완료 |
|                        |                        | 회원 상태별 필터링    | '전체 회원', '활성 회원', '만료 회원' 조건으로 목록만 분리해서 보기 제공  | 확인 완료 |
| **3. UI/UX 및 안정성** | **비동기 통신**        | Non-Freezing UI       | DB 쿼리 실행 시 화면 멈춤 방지를 위해 백그라운드(`SwingWorker`) 처리      | 확인 완료 |
|                        | **편의 기능**          | 입력 폼 초기화        | '초기화' 버튼 클릭 시 현재 작성 중이던 모든 입력 칸을 빈 상태로 되돌림    | 확인 완료 |
|                        | **예외 처리**          | 에러 팝업 핸들링      | DB 연결 실패, 쿼리 오류 등 문제 발생 시 사용자 친절 형태의 에러 팝업 제공 | 확인 완료 |
