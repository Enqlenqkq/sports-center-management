# 스포츠센터 회원 관리 프로그램 - 기능 명세서 (FS Sheet)

코드베이스 분석을 바탕으로 현재 프로그램에서 지원하는 모든 기능을 구체적으로 정리한 명세서입니다.

## 1. 회원 관리 (Member Management)

| 구분 | 주 기능 | 상세 기능 | 설명 | 상태 |
| :--- | :--- | :--- | :--- | :---: |
| **회원 관리** | **회원 등록** | 폼 기반 등록 | 이름, 연락처, 성별, 생년월일, 비고 등 필수/선택 정보를 입력하여 회원을 추가 | 완료 |
| | **회원 조회** | 전체 목록 표시 | 메인 화면 좌측 테이블에 전체 회원 목록을 표시 (비동기 로딩) | 완료 |
| | | 회원 검색 | '이름' 또는 '연락처'로 키워드 검색 후 결과를 테이블에 출력 | 완료 |
| | **회원 상세/수정** | 상세 창 및 정보 바인딩 | 회원 목록 더블클릭 시 상세 정보 다이얼로그 호출 및 텍스트 필드 채우기 | 완료 |
| | | 정보 업데이트 | 변경된 정보를 DB에 UPDATE하고 메인 화면 목록 새로고침 | 완료 |
| | **회원 삭제** | 레코드 삭제 | 확인 창(예/아니오) 후 DB에서 회원을 삭제하고 목록 최신화 | 완료 |

## 2. 강좌 관리 (Lesson Management)

| 구분 | 주 기능 | 상세 기능 | 설명 | 상태 |
| :--- | :--- | :--- | :--- | :---: |
| **강좌 관리** | **강좌 등록** | 폼 기반 등록 | 강좌명, 요일, 시간, 강사명, 정원, 가격을 입력하여 강좌 개설 | 완료 |
| | **강좌 조회** | 전체 목록 표시 | 메인 화면 우측 테이블에 강좌 목록을 비동기 처리하여 표출 | 완료 |
| | **강좌 수정** | 정보 바인딩 및 업데이트 | 강좌 목록 더블클릭 시 다이얼로그 호출 및 수정 후 DB 반영 | 완료 |
| | **강좌 삭제** | 레코드 삭제 | 강좌 삭제 버튼 클릭 시 DB 삭제 (수강 중인 회원이 있을 경우 제약) | 완료 |

## 3. 수강 관리 (Enrollment Management)

| 구분 | 주 기능 | 상세 기능 | 설명 | 상태 |
| :--- | :--- | :--- | :--- | :---: |
| **수강 관리** | **수강 내역 조회** | 회원별 수강 조회 | 회원 상세 창 내부 하단 테이블에 해당 회원이 수강 중인 강좌 목록 표출 (강좌명 Join) | 완료 |
| | **수강 신청** | 강좌 선택 및 기간 지정 | `[수강 신청]` 버튼을 통해 드롭다운으로 강좌 선택 및 시작일/종료일 지정 후 등록 | 완료 |
| | | 무결성 검증 | 이미 수강 중인 강좌 중복 신청 방지 및 정원(Capacity) 초과 방지 체크 로직 적용 | 완료 |
| | **수강 취소** | 수강 이력 삭제 | 수강 내역 테이블에서 선택 후 `[수강 취소]` 클릭 시 삭제 처리 | 완료 |

## 4. UI/UX 및 시스템 아키텍처

| 구분 | 주 기능 | 상세 기능 | 설명 | 상태 |
| :--- | :--- | :--- | :--- | :---: |
| **시스템** | **비동기 통신** | `SwingWorker` 처리 | DB 질의시 백그라운드 스레드를 사용하여 메인 UI Freezing 방지 | 완료 |
| | **디자인 시스템** | UI 룩앤필 적용 | `FlatLightLaf`를 적용하여 모던하고 일관된 테마 제공 | 완료 |
| | | 커스텀 컴포넌트 | `RoundedPanel`을 통한 둥근 테두리 및 그림자 효과로 시각적 향상 | 완료 |
| | **아키텍처** | MVC & DAO 패턴 | 각 로직을 분리하고 인터페이스를 사용하여 모듈 간 결합도를 최소화 | 완료 |
| | **환경설정** | Dotenv 연동 | `.env` 파일을 통해 DB 계정 및 비밀번호 등 민감한 정보를 안전하게 관리 | 완료 |
